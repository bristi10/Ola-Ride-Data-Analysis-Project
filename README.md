# 🚖 Ola Ride Insights
Interactive dashboard built using Streamlit and Power BI to analyze Ola ride data.

## 📌 Project Overview
Ola Ride Insights is a data analytics project focused on analyzing ride-sharing data to extract business insights.

## Live Demo app
Streamlit_link:https://ola-ride-data-analysis-project-xhwzhtcdzxa8wdv8hb4uuf.streamlit.app/

## 🛠️ Technologies Used
- SQL
- Python (Pandas, NumPy)
- Power BI
- Streamlit

## 🎯 Key Features
- Vehicle type filters
- KPI metrics
- Ride volume analysis
- Booking status breakdown
- Revenue insights
- Customer & driver ratings analysis
- Cancellation reason analysis
